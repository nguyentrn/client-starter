# NextJs Starter

## What is it included

**NextJs Starter** included apollo-client, redux-toolkit, styled-components, react-helmet

## How to use

### Download manually

Download the example:

```bash
git clone https://github.com/nguyentrn/client-starter.git
cd with-redux-toolkit
```

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
